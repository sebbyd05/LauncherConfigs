# LauncherConfigs
## What is this?
This is the open source repository for all servers and mods related to the X10 Launcher. All isses found with the mods should be put here (issues tab). This will be updated as frequently as possible.
## Which branch is the one currently deployed
At any given time the main branch is the one that is deployed to the file server, all other branches are just tests, or changes being staged.
